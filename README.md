# fundo
Fundo is a modern and responsive landing page designed to showcase a wide range of furniture products to users in an attractive and user-friendly way. The platform focuses on delivering a clean UI/UX experience that highlights different categories of furniture, including home, office, and decorative pieces.

The project emphasizes visual presentation, smooth navigation, and fast loading performance to help potential customers easily explore available products. Fundo is built with scalability in mind, making it easy to extend into a full e-commerce platform in the future.

Key features include a responsive design for all devices, well-structured product sections, engaging layouts, and optimized assets for performance and SEO. This project is ideal for furniture businesses looking to establish a strong online presence and effectively display their collections.
